# Порівняння алгоритмів пошуку підрядка

## Алгоритми:
- Боєра–Мура (BM)
- Кнута–Морріса–Пратта (KMP)
- Рабіна–Карпа (RK)

## Використані підрядки
- Реальний: `"використання"`
- Вигаданий: `"wer"`

## Результати 

### Текст 1
```
'BM': {'real': 0.004009042051620781, 'fake': 0.5608549999305978}, 
'KMP': {'real': 0.01269933395087719, 'fake': 0.5740932909538969}, 
'RK': {'real': 0.027138334000483155, 'fake': 1.6222315829945728}
```

**Найшвидший:** BM

### Текст 2
```
'BM': {'real': 0.007890500011853874, 'fake': 0.789965832955204}, 
'KMP': {'real': 0.024180165957659483, 'fake': 0.8157517919316888}, 
'RK': {'real': 0.05279270908795297, 'fake': 2.333342583035119}
```

**Найшвидший:** BM

## Висновок
Для обох текстів найшвидший алгоритм — Боєра–Мура.

КМП стабільний, але програє BM за часом.

Рабін–Карп — найповільніший, особливо для вигаданого підрядка.
